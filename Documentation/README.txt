Plugin Installation:

1) Upload the contents of the "Upload" directory into your forums root directory

2) Make sure NewPoints Plugin is installed already

3) Visit: ACP » NewPoints » Plugins and install the Buy Format Plugin

4) Visit: ACP » NewPoints » Buy Format and add your styles via "Add Style"

5) Visit: ACP » NewPoints » Buy Format to view your styles via "View Styles"	

** add new format styles.

6) Visit: ACP » NewPoints » Settings » Buy Format	to set Allowed Groups and or to turn plugin on or off without losing data
